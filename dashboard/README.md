## Monitoring Dashboard

This automated tracker is designed for stakeholders to continuously monitor traffic, funnel health and spot historical anomalies.

**Key Metrics Monitored:**
* **Traffic Source:** View what source the traffic is coming from.
* **Funnel Health:** Total Orders, Total Success Order, Drop-off by stage.
* **Acquisition Efficiency:** Bounce Rate, Cart Abandonment Rate, Conversion Rate.
* **Trend Analysis:** Month-over-Month (MoM) and Year-over-Year (YoY) performance trajectories.
